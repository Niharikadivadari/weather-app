# 🌦️ Weather App

A simple and responsive weather application that fetches real-time weather data using the OpenWeatherMap API. Users can search for any city and view key weather details instantly.

## Features

* Search weather by city name
* Displays temperature in Celsius
* Shows humidity and wind speed
* Dynamic weather icons based on conditions
* Error handling for invalid city names
* Loading indicator for better user experience
* Supports both button click and Enter key search

## Technologies Used

* HTML
* CSS
* JavaScript
* OpenWeatherMap API

## How It Works

1. User enters a city name
2. Application sends a request to the weather API
3. Receives data in JSON format
4. Updates UI with temperature, humidity, and wind speed
5. Displays corresponding weather icon

## Project Structure

* index.html → Main structure and logic
* style.css → Styling and layout
* weather-app-img/ → Weather icons and images
